# GO飯

  IoTハンガー。服をかけるだけで保温性を測定、行きたい場所をアプリに話しかけると適切な服を選びハンガーラックの中央に提示。

  ![](/img/2016/to.jpg)
  ![](/img/2016/to2.jpg)
  ![](/img/2016/to3.jpg)

### Description

  温度センサと発熱体を用いて服の保温性を自動で測定し、行きたい場所と時間をアプリケーションで入力すると、最適な服をレコメンドした上でハンガーラックの取りやすい位置に移動してくれるIoTハンガー。
  
  回路担当。
